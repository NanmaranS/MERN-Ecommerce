# 🛒 MERN E-Commerce Application

A full-stack e-commerce application built with the MERN stack featuring authentication, product management, cart & order system, and real-time search.

---

## 🌐 Live Demo

Frontend: https://mern-ecommerce-frontend-0jer.onrender.com
Backend: https://mern-ecommerce-back-j8ux.onrender.com

---

## 🚀 Features

* 🔐 User Authentication (Register & Login)
* 🔑 JWT Authentication with HTTP-only Cookies
* 🔒 Password Hashing for Security
* 🛍️ Product Management (CRUD operations)
* 🛒 Cart System (User-specific cart)
* 📦 Order Management (User-specific orders)
* 🔍 Product Search Functionality
* 📱 Fully Responsive UI

---

## 🧑‍💻 Tech Stack

**Frontend**

* React.js
* Axios
* CSS / Bootstrap

**Backend**

* Node.js
* Express.js
* MongoDB (Mongoose)

**Authentication**

* JSON Web Token (JWT)
* Cookies

---

## 📁 Project Structure

backend/ – API, database, authentication
frontend/ – React UI

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/NanmaranS/MERN-Ecommerce.git
cd MERN-Ecommerce
```

---

### 2. Setup Backend

```
cd backend
npm install
```

Create a `.env` file:

```
PORT=5001
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

Run backend:

```
npm start
```

---

### 3. Setup Frontend

```
cd frontend
npm install
npm run dev
```
